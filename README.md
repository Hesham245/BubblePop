# BubblePop

A Python-based hand-tracking arcade game built using:

- OpenCV
- MediaPipe
- Pygame

## Features
- Real-time hand tracking using your webcam
- Bubble popping mechanics
- Sound effects and background music
- Score tracking and leaderboard system
- Custom graphics and fonts

## Technologies Used
- Python
- Pygame
- OpenCV
- MediaPipe

## How to Run

1. Install Python 3.12
2. Install dependencies:
   pip install -r requirements.txt
3. Run:
   python main.py
